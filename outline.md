# IO性能测试管理平台 - 项目概述

## 项目目标
构建一个专业的前后端分离IO性能测试管理平台，支持节点管理、用例配置、任务执行、实时监控和权限管理。

## 核心功能模块

### 1. 用户认证与权限管理
- 用户注册/登录系统
- 角色权限控制（普通用户、管理员）
- 任务可见性设置（公开/私有）

### 2. 节点信息管理
- 节点CRUD操作
- 节点登录配置（用户名、密码、登录方式）
- 节点状态监控
- 分区信息管理

### 3. 用例信息管理
- FIO参数配置界面
- IO块大小、队列深度、压缩比设置
- IO数据量和时间配置
- 用例模板管理

### 4. 任务管理核心
- 任务创建与配置
- 节点选择与分区映射
- 用例关联与参数继承
- 任务生命周期管理（开始、结束、克隆）

### 5. 实时监控与数据分析
- IO抖动图表实时显示
- iostat日志收集与展示
- 性能指标可视化
- 历史数据查询

### 6. 任务空间管理
- 个人任务空间
- 任务分类与标签
- 搜索与筛选功能

## 技术架构

### 前端技术栈
- HTML5 + CSS3 + JavaScript
- Tailwind CSS（样式框架）
- ECharts.js（数据可视化）
- Anime.js（动画效果）
- WebSocket（实时通信）

### 后端技术栈
- Node.js + Express
- MySQL数据库
- JWT认证
- WebSocket服务

### 数据库设计
- 用户表、角色表
- 节点表、分区表
- 用例表、参数表
- 任务表、任务节点关联表
- 日志表、监控数据表
